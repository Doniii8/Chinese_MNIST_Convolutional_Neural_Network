# Chinese MNIST Convolutional Network
This is a project I made to get back into CNN's using MNIST's Chinese characters image dataset, where I used a Convolutional Neural Network in order to make a model that can accurately predict from 15 
characters (the classes). I found this dataset on kaggle: https://www.kaggle.com/datasets/gpreda/chinese-mnist?select=chinese_mnist.csv

Some of the basic libraries that I used were Numpy, Pytorch, OpenCV, and SKlearn.
